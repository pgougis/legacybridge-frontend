legacybridge-web — React SPA (Vite 6 + React 18 + TypeScript + React Router v6)
================================================================================

Stack
-----
  - Vite 6.4
  - React 18.3
  - React Router v6.28
  - TypeScript 5.6
  - CSS variables (no external UI library)
  - Vite proxy → backend http://localhost:5064

Dev
---
  cd legacybridge-web
  npm install
  npm run dev      → http://localhost:5173
  npm run build    → dist/

Role → Home
-----------
  Admin   → /admin/dashboard
  Manager → /manager/dashboard
  Member  → /member/sources
  Viewer  → /viewer

Auth
----
  JWT stored in localStorage under key "lb_token"
  Claims decoded from token: nameid → userId, customerId, role (ClaimTypes.Role)
  401 responses → automatic redirect to /login

Manager isolation
-----------------
  Manager sees only resources with OwnerManagerId = their userId (set on backend at creation)
  Manager cannot create users with Admin role
  Manager's customerId is injected automatically on create (from JWT claim)
  Dashboard shows isolation notice banner
